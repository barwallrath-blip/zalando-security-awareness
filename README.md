# Zalando Security Awareness Project

## Overview
This project was created as part of my Cybersteps cybersecurity journey.

The goal was to simulate a phishing awareness campaign for Zalando using Gophish and MailHog while learning how phishing attacks work from a defender's perspective.

## Objectives
- Build a phishing awareness campaign
- Create a phishing email template
- Design a phishing landing page
- Capture campaign statistics
- Understand user awareness and social engineering techniques

## Technologies
- Gophish
- MailHog
- Kali Linux
- HTML
- CSS

## Project Structure
📁 gophish/
- Email templates
- Landing pages
- Campaign configuration

📁 mailhog/
- Email testing

## What I Learned
- Creating phishing simulations in a safe lab environment
- Email delivery and testing
- Social engineering awareness
- Security awareness campaign planning
- Reporting campaign results
